# Dicas para um programador

<p align="center">
    Aqui temos dicas para programadores que estão com dificuldades com algum assunto ou app.
</p>

<details>
    <summary><strong>Git</strong></summary>
    <a href="https://github.com/Lumikinho/Dicas-para-um-programador/blob/main/Gitcomocolocar.md">
        Como colocar o meu código no github?
    </a>
</details>

<details>
    <summary><strong>Boas Práticas</strong></summary>
    <ul>
        <li>Mantenha commits pequenos e frequentes.</li>
        <li>Dê nomes claros para variáveis e funções.</li>
        <li>Organize o código e delete o que não precisa.</li>
    </ul>
</details>

<details>
    <summary><strong>Ferramentas</strong></summary>
    <ul>
        <li>VS Code, Prettier, ESLint, GitLens.</li>
        <li>Explore extensões que aumentam produtividade.</li>
    </ul>
</details>

<details>
    <summary><strong>Documentação</strong></summary>
    <ul>
        <li>Leia sempre a documentação de bibliotecas.</li>
        <li>Documente trechos importantes em Markdown.</li>
    </ul>
</details>

<details>
    <summary><strong>Contribuições</strong></summary>
    <ul>
        <li>Abra uma issue se encontrar problema.</li>
        <li>Pull requests são bem-vindos.</li>
    </ul>
</details>
